Getting Started with Git

  • Create an account with Github.
  • Set up your account by typing in the profile details.i.e profile name.
  • Navigate to the repositories tab.
  • Click on new to create a new clean repository.
  • Type your preferred name for the new repository.
  • Type a small description of your repository at the description section.
  • Select whether you want your repository settings to be private or public.(preferred: Public)
  • Select Add a README mark down file.
  • Click on create repository.
  • Click on your new repository.
  • On the Code tab which is green in color, click on it and copy the github HTTPS link.
  • Open your Git Bash application on your Desktop computer.
  • Navigate to Desktop using the command "cd Desktop".
  • Type "git clone" command and on the same line,paste the github HTTPS link
    to copy your remote central repository on github to the Desktop destination on your
    Desktop/Laptop.
  • Navigate to the working directory/local repository you just cloned on your
    Desktop by typing "cd <"name of local repository/working directory">.
  • Create a new file by typing the command "touch <"name of your file and its file extension respectively">.i.e index.html, home.html, register.html.
  • Type the command git add "name of file", to add it to the staging area.
  • Type the command git status to see the state of the files in the working directory.
    It lets you see which changes have been staged, which haven't, and which files aren't being tracked by Git.
  • Type the command git commit -m"message about your commit goes here" to save the changes that happened to the files in the local repository.
  • Type git push origin master to upload local repository content to a remote repository on github.

Congratulations on learning Git !
